# Open Source Policy

This project follows a simple open-source license policy.

## Allow
- MIT
- BSD-2-Clause
- BSD-3-Clause
- Apache-2.0
- MPL-2.0

## Warn (manual review)
- LGPL-2.1
- LGPL-3.0
- EPL-2.0

## Deny
- AGPL-3.0
- GPL-3.0 (unless an explicit written exemption is granted)

All dependencies must be pinned via `*.lock.txt` files with hashes.
